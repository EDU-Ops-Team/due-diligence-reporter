## H2: Gating Factors: critical unknowns that impact our timeline and costs

*Guidance: Gating Factors translate the unknowns from the Executive Summary into a decision framework. Each gate has: a name (what the unknown is), a resolution date (when you'll know), and two branches (what happens if good / if bad). Neeraj will trace each gate backwards — "what triggers this?" — so every gate must have a clear trigger.*

*Gates are numbered by resolution order (Gate 0 = resolves first). If an unknown is a potential site kill, it must be Gate 0.*
